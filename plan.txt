土木工事現場タスク管理システム 実装計画
土木工事現場における作業タスクを管理するWebベースのプロトタイプシステムを構築します。エクセルライクなUIで直感的な操作を実現し、作業データの柔軟な管理と表示カスタマイズを可能にします。

ユーザーレビュー必須事項
IMPORTANT

単一HTMLファイルでの実装 すべての機能（HTML、CSS、JavaScript）を1つのHTMLファイルに統合します。データはlocalStorageに保存され、外部ファイルやサーバーは不要です。

IMPORTANT

セルマージのレンダリング方式 作業データ項目ごとに「縦の数」が異なる場合、テーブルのrowspan属性を使用してセルマージを実装します。これにより、例えば「使用機械1, 2, 3」のように3つのデータを持つ項目は、他の項目が1つのデータしか持たない場合でも視覚的に整列します。

構造
HTMLマークアップ: タブUI、タスク表示テーブル、表示設定UI、モーダルダイアログ
CSSスタイル: モダンで洗練されたデザイン（グラデーション、アニメーション、レスポンシブ対応）
JavaScriptロジック: すべての機能実装
タスク表示画面の機能
タスク作成ボタン: クリックでテーブル最下部に新しい空のタスク行を追加
タスク表示テーブル:
左端固定列に削除ボタン配置
各列は表示設定画面で定義された項目に対応
セルは編集可能（contenteditable属性を使用）
複数データを持つ項目は改行区切りで表示し、rowspanでセルマージ
削除確認モーダル: タスク削除前に確認ダイアログを表示
表示設定画面の機能
項目リスト表示:
現在設定されている項目を視覚的に表示
各項目に編集・削除ボタンを配置
項目追加/編集フォーム:
項目名入力欄
縦の数（1つのタスク内でのデータ行数）入力欄
ドラッグ&ドロップ並び替え:
HTML5 Drag and Drop APIを使用
項目の視覚的な位置変更が即座にタスク表示テーブルに反映
データ管理
localStorage使用: ブラウザのlocalStorageにJSON形式でデータを保存
データ構造:
{
  columns: [
    { id: "col1", name: "担当会社", rows: 1 },
    { id: "col2", name: "使用機械", rows: 3 },
    ...
  ],
  tasks: [
    {
      id: "task1",
      data: {
        "col1": ["A社"],
        "col2": ["クレーン", "ショベルカー", "トラック"],
        ...
      }
    },
    ...
  ]
}

カラースキーム: 建設業界をイメージした青系のプロフェッショナルなデザイン
グラデーション: ヘッダーやボタンに洗練されたグラデーション適用
アニメーション: ホバーエフェクト、モーダルのフェードイン/アウト
