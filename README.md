アプリのプロトタイプを作ってほしい。
・土木工事現場のタスク管理システム
・何を使ってもいいが、最終的なアウトプットはhtmlファイル1つにしてほしい
・タスク表示画面と表示設定画面があり、画面内タブで表示切り替え可能
・タスク表示画面はエクセルライクな感じ
・タスク表示画面ではタスク作成ボタンが上部にあり、その下にエクセルライクなタスク表示テーブルがある
・タスク表示部は列が担当会社、作業内容などの作業データ項目、行が1作業を表す
・一番左の列には削除ボタン配置列が固定表示され、各作業の削除が可能
・作業削除時には確認モーダル画面が表示される
・作業作成するとテーブルの一番下にからっぽの作業が追加される
・作業データ項目には使用機械1,2,3のように縦に複数データが入ることもある（作業データ項目ごとに1作業内での項目数が異なることがあり、項目ごとに縦のセルマージでレイアウトを維持する必要がある）
・表示設定画面では実際の表示をイメージしやすいようなUIで、表示する項目を追加・編集できる
・表示設定画面で設定できる内容は項目名、縦の数
・表示設定画面では追加済の項目の表示場所（列の左右）をドラッグで良い感じに入れ替えることもできる
